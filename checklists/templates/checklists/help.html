Ajuda & Documentação – Listas de Verificação

Modelos Principais

  • ChecklistTemplate: nome, ano, disciplina, descricao, criado_em, atualizado_em
  • ChecklistItem: template (FK), descricao, criterios, ordem, contratualizado_em_conselho
  • ChecklistStatus: template (FK), aluno (FK), classe (FK), percent_complete, atualizado_em
  • ChecklistMark: status (FK), item (FK), estado, comentario, criado_em, marcado_por (FK), validacao_professor

Endpoints

  • /checklists/minhas/ — Listas do aluno (ativas/concluídas)
  • /checklists/<template_id>/ — Detalhe e marcação de objetivos
  • /checklists/turma/<class_id>/<template_id>/ — Visão coletiva (professor)
  • /ajuda/ — Esta página de ajuda/documentação

Scripts de Seed

  • seed_checklists_5ano.py — Cria template e itens de Português 5.º ano
  • seed_checklists_6ano.py — Cria template e itens de Português 6.º ano
  • Nota: só usar em dev/test. Aborta se já existir o template.

Instruções de Uso

  • Aluno: Marca objetivos, adiciona comentários, acompanha progresso.
  • Professor: Vê visão coletiva, valida/retifica, acompanha progresso coletivo e individual.

Testes Automatizados

  • Testes de criação/edição, permissões, notificações e seeds.
  • Para correr: python manage.py test checklists

Para mais detalhes técnicos, consulte o README.md ou contacte a equipa de desenvolvimento.
